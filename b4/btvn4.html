Câu 1: Bài tập về selectors(bộ chọn)

Yêu cầu: Tạo 1 thư mục đặt tên là CSS, trong thư mục CSS tạo file style.css, sau đó trong file index.html hãy link thêm file style.css vào (đường dẫn relative - tương đối). Tạo ra các đoạn văn bản dưới đây:

Tạo thẻ p như này và thêm class="color-red", sau đó viết thêm CSS để dòng chữ này thành màu đỏ (Dùng bộ chọn class - class selectors)

Tạo thẻ p như này và thêm id="color-green", sau đó viết thêm CSS để dòng chữ này màu xanh lá(Dùng bộ chọn id - id selected)
